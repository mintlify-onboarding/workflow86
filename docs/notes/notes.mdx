## What are notes?
Notes are objects you can place directly onto your workflow canvas to document information, processes, workflows and more.

<img src="/images/notes-demo.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

## What can I do with notes?
### Map out, diagram and brainstorm before you build
You can use notes to help map out a process or workflow before you start building it. Your team can mark out the individual steps of the process, connect the notes together to create a flow diagram, and then use this as the basis for the next step - building it out into an automated workflow on the same canvas.

<img src="/images/notes-brainstorm.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

### Comments, notes, instructions and other documentation
You can use notes to insert comments, descriptions, instructions or other forms of documentation for your workflows.


<img src="/images/notes-team.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

### Swimlanes
You can also use notes to create swimlanes that mark out which teams are responsible for which actions in a workflow

<img src="/images/notes-swimlanes.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

Or use them to record details for other team members or just to jot down ideas and thoughts.

## Working with notes
### Create a note
To create a note, click on the Notes button on the right hand side of the canvas.

<img src="/images/notes_1-6efd77bbc40b1764e16bf038c7ecd093 (2).png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

A menu of different types of notes will appear, and you can then drag and drop the note type you want to use onto the canvas

<img src="/images/notes-drag-and-drop-36a113a1ddcb045b7240cc3b976a12e4.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

### Edit a note
To edit a note, double click into into the note or click the edit button.

<img src="/images/notes-team-dd45273d27eebe6905f0100a0a7f37a6 (1).png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

### Move a note around
To move a note around, click to select the note and then drag it to the desired position on the canvas.

<img src="/images/download (1).png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

### Save a note
To save your edits, click the "Save Edits" button. Notes are also auto-saved when you click out of the note or exit the edit mode.

<img src="/images/download (2).png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

<Info>Make sure to save any edits you made a note before exiting the canvas</Info>

## Writing notes
Notes use the same rich text editor as the Read Me, meaning you can insert a variety of content and format it as well. To format content inside of a note, simply highlight it to bring up the formatting menu

<img src="/images/notes_6-d67f3c65af3eb1520fbfe91c306d7da0.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

You can insert tables, images, create hyperlinks, and even embed videos

<img src="/images/notes-full-dec111c10b4ba32b71c4582703963be3.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

## Show/hide notes on the canvas
Notes are displayed and visible by default, but you can toggle to hide them from being displayed on the canvas via the show/hide notes button here on the canvas control panel.

<img src="/images/show-hide-button-5e6cc682e593c9cef9c13af4ca8c8464.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

Click the button to hide all notes on your canvas. This will not delete or modify your notes in any way, it just hides them from view on the canvas so they are out of the way/not visible to any users on the canvas.

<img src="/images/notes-visible-0df4dccba4e10d89321bd3502948258b.png" style={{
  border: '1px solid #dfdfdf',

## Map a Zap
Map a Zap allows you to auto-generate documentation for all the steps in a zap using the Workflow86 notes feature.

<img src="/images/map-zap-result-0a60a9c165fd3f90fb32f477fc440bb0.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

### Share a Zap
To map a zap, we firstly need to share it and obtain a link to a zap. Go to Zapier and the zap you want to map. If it has not been turned on or published yet, turn it on and then click Copy Link when presented with the dialog below:

<img src="/images/share-zap-189d7a8aefe93b1a367eb4e2440eea65.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

You can also press the Share this Zap button at the top right corner

<img src="/images/share-zap123123.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

Then copy the link presented in the dialog:

<img src="/images/share-zap3-ecc6c5e3e38ec829be785eafec66f4c5.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

### Map a Zap in Workflow86
Open the notes menu and then drag and drop the Map a Zap module onto the area of the canvas where you want to place the notes.

<img src="/images/map-zap-19701468cf4f190bc7083db14dc01840.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

A dialog will then appear where you will copy and paste the URL you got in the previous step. Note that you can copy and paste in multiple zaps at once.

<img src="/images/map-zap-dialog-335b33ec0281cb08c44e86fa83299711.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

Click Map It to proceed. Once completed, all steps in your zaps will now be rendered as individual notes in the same order and sequence as your zap:

<img src="/images/map-zap-result-0a60a9c165fd3f90fb32f477fc440bb0 (1).png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />

Each note contains the step number, name, description and parameters associated with each step in your zap. You can edit or add further notes by double clicking into these notes.

<img src="/images/map-zap-notes-7512b3493519ac030f5cb12b49387278.png" style={{
  border: '1px solid #dfdfdf',
  borderRadius: '10px'
}} />
