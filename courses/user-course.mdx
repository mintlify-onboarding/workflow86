## Workflow User Course

This course covers how to use your Workflow86 account to start automated workflows and complete assigned tasks as part of workflows.

## Who this is for

This course is intended for users who will be interacting with workflows they have built, or have been built by others. It is relevant for all Workflow86 users including Admins, Editors and End-users.

## What you will learn

After completing this course, you will be able to:

- Start automated workflows;
- Complete tasks assigned to you as part of a workflow.

## How long will it take

Completing this course should take about 15 minutes.
